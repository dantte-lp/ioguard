# Support

## Getting Help

- **Documentation**: See [docs/en/](docs/en/) for English, [docs/ru/](docs/ru/) for Russian
- **Discussions**: Use [GitHub Discussions](https://github.com/dantte-lp/ringwall/discussions)
  for questions and general discussion
- **Issues**: Use [GitHub Issues](https://github.com/dantte-lp/ringwall/issues) for bug reports
  and feature requests

## Before Opening an Issue

1. Check existing [issues](https://github.com/dantte-lp/ringwall/issues) and
   [discussions](https://github.com/dantte-lp/ringwall/discussions)
2. Include your OS/kernel version, compiler version, and build configuration
3. Provide minimal steps to reproduce the problem
4. Include relevant log output

## Security Issues

For security vulnerabilities, see [SECURITY.md](SECURITY.md).
Do not open public issues for security problems.
